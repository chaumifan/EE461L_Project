    About Pocket Recipes


    Have you ever had the problem where you have a full fridge but don't know what to do with all your ingredients?
    Simply enter in all of the ingredients you want to use and we will curate a list of recipes that fit your fridge.

    Created by: Calvin Ly, Evonne Ng, Kevin Chau, Hasan Genc, Matt Tawil


    It's as easy as 1, 2, 3!


    1. Enter all ingredients you want to include in the recipes in the TOP box

    2. Enter all ingredients you are allergic to or just don't like in the BOTTOM box

    3. Submit Query!


    Tools used to create this project

    Google Cloud SDK written in Python 2.7
    Bootstrap and JQuery for HTML, CSS, and Javascript
    SQL for database storage and management
    Testing
      Selenium for UI testing
      Controller tests with integration testing (REST)
      Python for database queries (Component and Unit testing)
